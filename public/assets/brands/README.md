# Brand Logos

Place brand logo files in this directory with the following naming convention:

- File format: PNG (recommended)
- Naming: `{brand-id}.png` where brand-id matches the ID from segments-brands.yaml
- Size: 100x100px minimum (will be scaled to fit)
